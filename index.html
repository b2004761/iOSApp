Cách cài trên iPhone

  1. 1Mở trang này bằng Safari trên iPhone.
  2. 2Chọn bản V9 muốn dùng và cho phép tải profile.
  3. 3Vào Settings, chọn Profile Downloaded, rồi bấm Install.
  4. 4Nếu chỉ muốn lưu file về máy, dùng nút Tải file trong từng thẻ cấu hình.
  5. 5Muốn đổi giữa JS Off và JS On thì gỡ profile cũ trước, sau đó cài bản mới.

Lưu ý quan trọng

Profile này rất mạnh. Một số tính năng như cài hoặc gỡ app, iCloud sync, Siri, Game Center, quay màn hình, Safari JavaScript ở bản JS Off, vị trí, AirPlay, Apple Watch và một số dịch vụ nền sẽ bị hạn chế cho đến khi bạn gỡ profile.
UltimateMax V9 không thể ép xung, tăng FPS trực tiếp, đổi tần số quét hay điều khiển phần cứng. Profile chỉ giảm tải gián tiếp qua các chính sách iOS cho phép.
